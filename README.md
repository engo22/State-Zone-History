# 📜 История State Zone (СЗ)

Полная хроника Discord-сообщества **State Zone** (СЗ) с 2022 по 2026 год.

Проект пережил:
- взлёты и падения,
- конфликты и интриги,
- выборы, перевороты, крахи серверов,
- войну за Новокузнецк на PixelPlanet,
- экспансию в сообщество Nations RP,
- рождение «мигрантов» и Парламента.
